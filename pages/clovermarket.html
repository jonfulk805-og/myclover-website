🛠 Build & Publish

From Idea to Marketplace in 4 Steps

The Creator SDK handles everything. Scaffold, build, test, publish — all from the command line.

1

Scaffold

cloverstack-ctl sdk init — pick a template, answer prompts, get a full project with manifests, Docker config, and wizard.

2

Build

Write your app in any language. Use the CloverStack SDK to integrate with AI, database, auth, and more. Hot-reload dev mode included.

3

Test & Validate

cloverstack-ctl sdk validate --all — manifest checks, Docker build, health checks, security scan, wizard test. All automated.

4

Publish

cloverstack-ctl sdk submit — auto-scan + AI review. Most apps approved in under 15 minutes. Set your price, go live.

# Scaffold a new app
$ cloverstack-ctl sdk init --template react-dashboard
  ? App name: TicketDesk
  ? App ID: com.myname.ticketdesk
  ? Category: business
  ✓ Created ticketdesk/

# Develop with hot-reload
$ cloverstack-ctl sdk dev
  🌏 App running at http://localhost:8080
  👁 Watching for changes...

# Validate everything
$ cloverstack-ctl sdk validate --all
  ✓ cloverapp.yml — valid
  ✓ Docker build — passes
  ✓ Health check — 200 OK
  ✓ Security scan — clean
  ✓ All 10 checks passed

# Ship it
$ cloverstack-ctl sdk submit
  ✓ Submitted — Review ID: REV-2026-00142
  ✓ Auto-approved in 12 minutes. Live now!
