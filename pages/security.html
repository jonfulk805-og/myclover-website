🐳 Deploy

One Command. Full Stack.

Docker + Portainer. Run any weapon individually or deploy the full Arsenal with a single docker-compose file. No CloverOS required — works on any Linux, Mac, or Windows with Docker.

Individual Weapons

Pull and run any product standalone.

# Overwatch — Network Monitoring docker pull myclovertech/netmon:latest docker run -d --name netmon -p 8080:8080 myclovertech/netmon # Blackbox — SIEM & Log Intelligence docker pull myclovertech/sentrylog:latest docker run -d --name sentrylog -p 8081:8081 -p 514:514 myclovertech/sentrylog # Claymore — Honeypot Deception docker pull myclovertech/cloverstinger:latest docker run -d --name cloverstinger -p 8082:8082 -p 2222:2222 myclovertech/cloverstinger
