Setup

Running in 5 Minutes. Seriously.

No complex infrastructure. No Docker required. Just Python, an API key, and you're live.

Install Dependencies

pip install -r requirements.txt

Python 3.9+ and about 30 seconds of patience.

Add Your API Key

Copy .env.example to .env, paste your OpenAI key. That's the whole config.

Start the Server

python main.py

Opens on port 8400. Knowledge base auto-ingests on first boot.

Chat With CloverBot

Open localhost:8400/widget and start a conversation. When ready, embed on your site with one tag.

<!-- Add this before </body> on any page --> <script src="https://your-server:8400/static/cloverbot.js" data-api-url="https://your-server:8400"> </script>

Self-Hosted AI

Production

Deploy It For Real

🔐

Reverse Proxy

Put nginx, Caddy, or Traefik in front for SSL termination and domain routing.

🔄

Process Manager

Use systemd, PM2, or supervisor to keep CloverBot running and auto-restart on failure.

🛡️

Auth on Admin

Add API key or basic auth to /api/conversations and /api/leads.

📡

CORS Lock

Update ALLOWED_ORIGINS to only your production domain. No wildcards.
